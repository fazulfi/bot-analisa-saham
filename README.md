# Bot Analisa Saham (Indonesia)

Deskripsi singkat:
- Bot ini menghasilkan sinyal BUY otomatis (dengan TP/SL) untuk saham Indonesia.
- Bahasa: Python
- Output: CSV per ticker, notifikasi via Telegram.

## Struktur awal
- src/               # kode utama
- data/              # CSV per ticker
- signals/           # penyimpanan sinyal per ticker
- tests/             # unit tests
- docs/              # dokumentasi

## Mulai
1. Clone repo
2. Buat virtualenv
3. Install dependencies (requirements.txt)
